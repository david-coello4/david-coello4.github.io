3D Print Art

Minecraft is a popular game where the user has the ability to create anything using a variety of blocks and their imagination. So I wanted to start a project where I built something in Minecraft, export the structures code into a 3D printing software, and 3D printing the structure. To start this project, I used Minecraft as a tool to create my structure that I wanted to 3D print, which is Spongebob and Patrick, SpongeBob being smaller compared to Patrick who is giant referencing a scene from the first Spongebob Movie scene.


Next, I extracted the files from Minecraft and transferred them into the software FlashPrint 5 and preparing to start the 3D printing process. So these images showcase the in-game version of the structure and the coded version of the structure in the FlashPrint 5.


After processing the files, I first created a draft print, the blank SpongeBob figure, to see how it printed in order to figure out if it came out as planned or if it needed adjustments. Once I felt confident about the structure of it, I finally printed out the entire build.


Unfortunately, the SpongeBob that's part of the Patrick build broke off because of the weak supports, so I had to use the first drafted SpongeBob as a replacement. After that, I painted each with their appropriate colors and finished this project
