Glitchy Bottom

Created in 2024

For this art work, I managed to glitch three different photos, the first two images are Doodle Bob and Doodle Squidward from the TV series Spongebob Squarepants. I managed to glitch these images by going into the code of each original image; using TextEdit, I scramble around the code to "glitch" the image, the process known as "Data Bending".



The third image was Doodle Patrick from the same TV series, however it looks different because I "glitched" the image differently by using an audio software named Audacity. Where I used the code of the image, insert it into a spectrogram, and played around with it until the final image became "glitched", the process know as "Data Moshing".
