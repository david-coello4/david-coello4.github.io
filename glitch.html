Glitch Art

For the first two images that look similar to each other, I went into the code of each original image and I use TextEdit to scramble around the code to make "glitch" the image, which are considered "Data Bending". The third image that looks differently than the others, I used Audacity where we used the code of the image, made it into a spectrogram, and played around it until the final images "Glitched", which is "Date Moshing".
