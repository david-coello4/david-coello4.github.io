Glitchy Bottom

Created in 2024

For this art work, I managed to glitch three different photos, the first two images of Doodle Bob and Doodle Squidward from the TV series Spongebob Squarepants. I managed to glitch these images by going into the code of each original image and I used TextEdit to scramble around the code to "glitch" the image, which I did by "Data Bending" the code of the image.



The third image of Doodle Patrick looks different because I "glitched" the image differently by using the software Audacity, where we used the code of the image, inserted it into a spectrogram, and played around with it until the final image became "glitched", which is called "Data Moshing".
