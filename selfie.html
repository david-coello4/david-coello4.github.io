Cybernetic Infectious Agent

Created in 2025

In this P5JS face sketch, my goal was to create a face using a variety of shapes such as triangles, ellipses, rectangles, quads, and lines; as well as how to reposition each shape and fill them with color. In order to make my face more interactive, I learned the use of If Statements, where if the mouse gets near the eyes, they change shapes. The Loop Function allows something to loop, as seen in the background as the glitchy lines. Lastly, I used a random function with MouseX and MouseY to make the "error window" image to reposition every frame while following the mouse curser to add to the glitchy effect.

To interact with the face, moving your mouse cursor over the eyes changes their form, at the same time, multiple 'error windows' follow the cursor around the screen, creating a chaotic and glitchy effect.
